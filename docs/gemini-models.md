# Gemini API — шпаргалка по моделям

> Дата составления: 27 мая 2026
> Источник: AI Studio + офиц. дока ai.google.dev
> Под задачу: редактирование изображений (image-to-image) для оформления групп ВК

---

## ⚠️ Важно про имена

В интерфейсе AI Studio показываются **отображаемые имена** (типа "Nano Banana 2").
В коде нужны **API-идентификаторы** (типа `gemini-3.1-flash-image-preview`).
Это разные вещи — не перепутай.

**Как сверить точное рабочее имя под свой ключ:** дёрни эндпоинт списка моделей
(`GET /v1beta/models`) — он вернёт ровно те ID, что доступны твоему ключу.
Особенно актуально для `-preview` моделей: их имена включают суффиксы/даты и
иногда меняются. Доке доверяй про возможности и цены, `list_models` — про точное имя.

## 🖼️ IMAGE-генерация (то, что тебе нужно)

Это вся линейка "Nano Banana", от слабой к мощной:

| Отображаемое имя | API-идентификатор             | Что это                          | Цена/картинка     |
|------------------|-------------------------------|----------------------------------|-------------------|
| Nano Banana      | `gemini-2.5-flash-image`      | старая базовая, быстро/дёшево    | ~$0.039           |
| Nano Banana 2    | `gemini-3.1-flash-image-preview` | новый дефолт, баланс цена/качество | $0.03–0.06 (по разрешению) |
| Nano Banana Pro  | `gemini-3-pro-image-preview`  | флагман, reasoning + 4K + текст  | дороже, +thinking-токены |

### Различия по делу

- **Nano Banana (`gemini-2.5-flash-image`)** — то, с чего ты начинал.
  Самая простая, отсюда и слабый результат. Для черновых тестов норм.

- **Nano Banana 2 (`gemini-3.1-flash-image-preview`)** — текущий дефолт в браузере.
  Заметно лучше базовой, при этом дёшево. Хороший рабочий вариант.

- **Nano Banana Pro (`gemini-3-pro-image-preview`)** — флагман.
  Использует "Thinking" (reasoning), лучше всех рендерит ТЕКСТ внутри картинки,
  тянет 4K и сложные макеты. **Для оформления ВК с названиями/текстом — лучший выбор.**
  НО: дороже в разы, и thinking-токены тоже тарифицируются → цена за генерацию
  непредсказуема (может быть $0.15–0.25, а не $0.04).

### Рекомендация под твою задачу (оформление групп ВК)
Текста в макетах много → бери **Nano Banana Pro** (`gemini-3-pro-image-preview`).
Если Pro дороговат для объёма → откатывайся на **Nano Banana 2** (`gemini-3.1-flash-image-preview`).
Базовую `gemini-2.5-flash-image` оставь только для быстрых проверок.

### Прочие медиа-модели (на будущее, не для image-edit)
- **Imagen 4** — text-to-image (генерация с нуля, без входной картинки), до 2K.
- **Veo 3.1 / Veo 3.1 Lite** — генерация видео.
- **Lyria 3** — генерация музыки.

## 🔤 Соглашение об именах версий

- **Stable** — фиксированная стабильная версия, не меняется. Для продакшена.
  Пример: `gemini-2.5-flash`
- **Preview** — превью, можно в продакшен, но более жёсткие лимиты,
  деприкейтят с уведомлением минимум за 2 недели. Пример: `gemini-2.5-flash-preview-09-2025`
- **Latest** — алиас на свежак (stable/preview/exp), хот-свопится при релизах.
  Пример: `gemini-flash-latest`
- **Experimental** — для фидбэка, НЕ для продакшена, жёсткие лимиты, нестабильно.

Совет: для своего парсера на текст бери **stable** (`gemini-2.5-flash`),
а для картинок придётся жить с **preview** (других image-моделей нового поколения нет).

## 💵 Контроль расхода (твой кейс: prepaid $15, auto-reload OFF)

- 1 image-генерация на flash ≈ **$0.04** → ~350 картинок на балансе
- 1 генерация на Pro ≈ **$0.15–0.25** (из-за thinking) → ~60–100 картинок
- Смотри `usageMetadata` в ответе:
  - `candidatesTokensDetails` с ~1290 IMAGE-токенов = 1 картинка
  - у Pro появится `thoughtsTokenCount` — он ТОЖЕ в счёт
- Запросы с ошибкой 400/500 НЕ тарифицируются (но жрут квоту)
- Баланс кончится → запросы встанут, в минус уйти нельзя (auto-reload off)
